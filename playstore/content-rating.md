# Play Console — Classification du contenu (IARC) : réponses

> À recopier dans Play Console → Contenu de l'application → Classification du contenu.
> Type d'application : **Utilitaire / Productivité / Professionnel** (pas un jeu).

Répondre **NON** à toutes les questions du questionnaire :

| Thème | Réponse |
|---|---|
| Violence (réaliste, fantastique, sang) | Non |
| Contenu sexuel / nudité | Non |
| Langage grossier | Non |
| Contenu effrayant / horreur | Non |
| Drogues, alcool, tabac | Non |
| Jeux d'argent / simulations de jeux d'argent | Non |
| Achats intégrés | Non |
| Partage de position de l'utilisateur | Non |
| Interaction entre utilisateurs / contenu généré par les utilisateurs | Non |
| Partage d'informations personnelles avec des tiers | Non |
| Accès non filtré à Internet (navigateur intégré) | Non |

**Classification attendue** : « Tout public » (PEGI 3 / ESRB Everyone).

**Public cible** : adultes / professionnels (**18 ans et plus**) — ne cible pas les enfants.
